## Install

```sh
npm install --save har-expander
```

## Usage

```

  Usage: har-expander [options]

  Options:

    -h, --help     output usage information
    -V, --version  output the version number

```

## API

### har-expander()

```js
var har-expander = require('har-expander')

har-expander()
```
